## 2.3 最小错误率分类

定义“0-1损失函数”：
λ(αi|wj) = { 0, i = j; 1, i != j. }, i,j = 1, ..., c。
该损失函数对应的条件损失为：
R(αi|**x**) = Σ(j=1, c)λ(αi|wj)P(wj|**x**)
            = Σ(j!=i)P(wj|**x**)
            = 1 - P(wi|**x**)
那么为了最小化损失率，我们的决策规则为：
选择 wi 当 P(wi|**x**) > P(wj|**x**)，对于所有的 j != i。

如图2.3所示，为 p(x/w1)/p(x/w2) 的似然率。一般来说该值的范围在0到无穷大之间。
